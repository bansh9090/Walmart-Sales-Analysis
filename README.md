# Walmart Sales Analysis

## Overview

This project focuses on analyzing Walmart sales data to understand sales patterns, compare store performance, and identify the factors that influence weekly sales.
Along with data analysis and visualization, I also explored basic machine learning models to understand how sales can be predicted using historical data.

## Objectives

* Clean and prepare raw sales data
* Perform SQL-based business analysis
* Create an interactive Power BI dashboard
* Identify top-performing stores and departments
* Analyze monthly and yearly sales trends
* Build basic machine learning models for sales prediction

## Dataset

The dataset contains information about Walmart stores, including:

* Store
* Department
* Date
* Weekly Sales
* Holiday Flag
* Temperature
* Fuel Price
* CPI
* Unemployment
* Store Type
* Store Size
* MarkDown1–MarkDown5

## Machine Learning

To understand how sales prediction works, I trained and evaluated two basic machine learning models using the cleaned dataset.

### Linear Regression

* Built a baseline regression model for predicting weekly sales.
* Used numerical features such as Temperature, Fuel Price, CPI, Unemployment, and Store Size.
* Evaluated the model using regression metrics.

### Random Forest Regressor

* Trained a Random Forest model to capture non-linear relationships.
* Compared its performance with Linear Regression.
* Observed that Random Forest produced better prediction accuracy on the dataset.
